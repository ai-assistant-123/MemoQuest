# MemoQuest - 沉浸式三级输出记忆工具

## 1. 项目简介
MemoQuest 是一款基于脑科学“三级输出”（Three-Level Output）记忆原理开发的辅助背诵 Web 应用。该应用通过游戏化的界面设计和渐进式的文本隐藏算法，帮助用户通过“输出倒逼输入”的方式，快速、高效地记忆长段文字或文章。

核心理念是将背诵过程转化为通关游戏，降低背诵的枯燥感，提高大脑的兴奋度和记忆效率。

## 2. 核心功能需求

### 2.1 文本输入 (Input Stage)
*   **文本编辑器**：提供一个大面积的输入框，允许用户粘贴任意想要背诵的中文或外文段落。
*   **示例加载**：提供一键加载经典示例（如《滕王阁序》），方便新用户快速上手体验。
*   **智能分词准备**：系统底层需支持中文智能分词，确保后续的游戏环节不会将完整的词语强行拆散。

### 2.2 记忆游戏 (Game Stage)
游戏分为三个难度等级，对应“三级输出”的三个阶段：

*   **第一级：字词间隔隐藏 (Level 1)**
    *   **算法逻辑**：以词语为单位，利用 `Intl.Segmenter` API 进行分词，采用**确定性交替隐藏策略**（显示一个词，隐藏一个词）。
    *   **目标**：通过保留的字词引导回忆，建立初步的文本脉络。
    *   **特点**：非随机隐藏，无需刷新，保证背诵节奏的稳定性。

*   **第二级：句末隐藏 (Level 2)**
    *   **算法逻辑**：以标点符号（逗号、句号等）为界，保留每句开头的前 1 个词，隐藏该句剩余的所有内容直到下一个标点。
    *   **目标**：强化对句子结构的记忆，训练从句首提示词推导整句的能力。

*   **第三级：段落隐藏 (Level 3)**
    *   **算法逻辑**：仅保留每个段落的第一个字（或词），隐藏段落内其余所有内容。
    *   **目标**：最高难度的回忆训练，仅靠段首提示背诵全段，达成完全记忆。

### 2.3 交互机制
*   **点击揭示 (Click to Reveal)**：所有被隐藏的文字默认显示为游戏风格的占位符（如 "X"）。用户点击任意占位符即可“偷看”该处的正确文字，再次点击不隐藏（或保持显示），方便查漏补缺。
*   **原文对照 (Reference Mode)**：提供“查看原文/隐藏原文”的全局开关，允许用户在卡壳时快速扫视全文，而不需要重置游戏状态。

## 3. UI/UX 设计风格

### 3.1 视觉主题
*   **风格**：复古游戏风 (Retro/Arcade) 结合赛博朋克元素。
*   **配色**：
    *   背景：深色系 (#111827, #1f2937) 以减少视觉疲劳。
    *   强调色：青色 (Cyan) 用于引导，粉色 (Pink) 用于标点/高亮，靛蓝 (Indigo) 用于隐藏块。
    *   字体：标题使用像素风字体 ('Press Start 2P')，正文使用等宽字体 ('Roboto Mono') 保证字符对齐。

### 3.2 响应式布局
*   适配桌面端与移动端，确保在手机上也能流畅进行背诵训练。
*   文本区域自定义滚动条，保持界面沉浸感。

## 4. 技术栈

*   **核心框架**: React 19
*   **构建工具**: Vite (假设环境)
*   **语言**: TypeScript
*   **样式库**: Tailwind CSS
*   **图标库**: Lucide React
*   **关键 API**: `Intl.Segmenter` (用于浏览器端原生中文分词)

## 6. 使用指南

1.  **输入**：将需要背诵的文案复制到输入框中，点击“开始记忆”。
2.  **挑战 Level 1**：看着屏幕上断断续续的文字，尝试朗读全文。遇到想不起来的词，点击 "X" 查看，或者按住“查看原文”按钮。
3.  **挑战 Level 2**：当 Level 1 比较熟练后，点击顶部“第 2 级”。此时只有句首可见，难度升级。
4.  **挑战 Level 3**：最终挑战，仅保留段首字。如果你能看着这几个字背诵全文，恭喜你，记忆完成！
